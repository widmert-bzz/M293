LU1 HTML und Werkzeuge:

HTML= HyperText Markup Language, HTML seit 2014 HTML 5.
Element= Bausteine von HTML, werden durch Tag markiert.
Tag= (deutsch: Etikette, Marke, Markierung).
Attribute= z.B: <html lang='de'> / das attribut ist : lang='de'.
CSS= Cascading Style Sheets.
Hello World Ausgeben: 

      <!DOCTYPE html>
      <html lang='en'>
        <head>
          <meta charset='UTF-8'>
          <title>Title</title>
        </head>
        <body>
          Hello World
        </body>
      </html>

LU3 Elemente um HTML zu strukturieren:

Block-Elemente wie (Block-elemente = Strukturieren der Seite):
  • Absätze --> <p>
  • Listen: ungeordnete (mit Aufzählungspunkten) --> <ul> oder geordnete Listen (mit Zahlen) --> <ol>
  • Überschriften: von der 1. Ebene --> <h1> bis zur 6. Ebene --> <h6>
  • artikel --> <artikel>
  • abschnitte --> <section>
  • blockelement --> <div>
Inline-Elemente wie (inline-Elemente = Hervorhebung von gewissen texten):
  • Links --> <a>
  • hervorgehobene Wörter --> <em>
  • wichtige Wörter --> <strong>
  • kurze Zitate --> <q>
Oder Darstellungs elente wie --> <strong> (you can't put a <p> tag inside of <strong></strong>)
IMG darstellen, z.B: <img src='meinBild.jpg' alt='Ein Bild von mir' width='200' height='200'>
Natürlich darf nicht vergessen werden, dass es für all diese auch einen schliess tag gibt, z.B. </strong>

LU5 HTTP und Adressen:

Pfade:

HTTP --> Hypertext Transfer Protoco
HTTP-Mezhoden --> Get, Post, Put
HTTP-Statuscode:

Status Bedeutung Erklärung
200 OK Die Anfrage wurde erfolgreich verarbeitet.
302 Moved Permanently Hat funktioniert, die Zieladresse hat sich jedoch verändert.
304 Not modified Die Ressource hat sich seit dem letzten Zugriff nicht verändert. Der Client kann die Version aus seinem Cache verwenden.
404 Not found Ressource nicht gefunden.
500 Internal Server Error Server fehler.

Absolute adressen --> von überall aus erreichbar, z.B https://it.bzz.ch/wikiV2/_media/howto/pycharm/props01.png
relative adresse --> Funktioniert nur lokal, z.B ./img/props01.png
./ --> um raus zu gehen
/ordnerName --> um in einen ordner rein zu gehen

LU6 Formularelemente:


Grundlegender Aufbau:
  • ganz Aussen --> <form action='Wohin nach abschicken' method='wie daten übergeben werden'> </form>
  • in dem form-tag --> jegliche eingabe felder
  • untem im form-tag--> <button type='submit'>Absenden</button>
  • untem im form-tag, nach bedarf --> <button type='reset'>Zurücksetzen</button>
Eingabe Felder:

      <input id='xy' name='xy' type='text'>
      <input id='xy' name='xy' type='number'>
  • name --> Wird benötigt um daten weiterzuleiten
  • type --> gibt an was eingegeben werden kann
  • zusätzliche-Attribute --> minlength, required, max
Label + Inputfeld:

      <label for='email'>Email</label>
      <input id='email' name='email_address' type='text' />
Sieht dann so aus:


Textarea

      <textarea id="xx" name="xx" rows="4" cols="50">  </textarea>
Alles zusammen:

      <form action='./index.php' method='post'>
      <label for='email'>Email</label>
      <input id='email' name='email' type='text' minlength='10' required />
      <button type='submit'>Absenden</button>
      <button type='reset'>Zurücksetzen</button>
      </form>
